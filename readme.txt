Adds the possibility to set a republish date to pages, posts and custom post types.

== Description ==

The plugin adds a datetime box to your edit pages to set a republish date to posts, pages, and custom post types.

After you set a republish date the post will we updated at the given time.
The plugins works with posts, pages and custom post types

== Installation ==

1. Upload the 'gtuk-republish-posts' plugin to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= The post has not been updated =

Make sure you set the correct timezone.

= Can i add the republish date my custom post types =

The plugin works with your custom post types out of the box.
